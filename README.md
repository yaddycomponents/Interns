# Intern Projects

## For Interns

Your folder is in `interns/your-name/`. Just edit:

```
interns/your-name/
├── index.html
├── style.css
└── script.js
```

Push your changes:
```bash
git pull
git add .
git commit -m "Update my project"
git push
```

---

## For Admin

Any folder inside `interns/` is auto-detected on deploy.

```bash
npm run build   # Test locally
npm run dev     # Build + serve
```

Deploy: **Actions → Deploy to Vercel → Run workflow**
